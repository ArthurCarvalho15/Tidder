APIs que Estão Ajudando no Combate do Corona Vírus

No ano de 2019, um vírus se espalhou por toda a China e nesse momento se espalha por todo o mundo. A Organização Mundial da Saúde (OMS) já declarou que o surto da doença constitui uma Emergência de Saúde Pública de Importância Internacional e também como uma pandemia. Foram confirmados no mundo mais de 5 milhões de casos de COVID-19 e mais de 330 mil mortes até o momento. As medidas de proteção devem ser feitas como: lavar as mãos frequentemente com água e sabão, usar álcool em gel e cobrir a boca com o antebraço quando tossir ou espirrar (ou use um lenço descartável e, após tossir e/ou espirrar, jogue no lixo e lave as mãos). Se uma pessoa tiver sintomas menos agravantes, como tosse leve ou febre leve, geralmente não há necessidade de procurar atendimento médico. O ideal é ficar em casa, fazer autoisolamento e monitorar os sintomas. Procure atendimento médico imediato se tiver dificuldade de respirar ou dor/pressão no peito. Caso queira saber mais sobre o COVID-19 e como se previnir, entre no site do Ministério da Saúde para saber mais.

Sendo assim, como muitas APIs foram criadas e/ou melhoradas para ajudar a combate-lo, decidimos demonstrar abaixo algumas dessas principais APIs.

P.S. A descrição de cada API possui uma curiosidade sobre o COVID-19.

IBM WATSON

O Watson já cobre cerca de uma dúzia de APIs diferentes que ajudarão você a entender imagens, som e texto. Algumas dessas APIs são: a API de reconhecimento visual, a Tone Analyzer API e a API Explorer do Watson. Você insere seu conjunto de treinamento e, em seguida, a API aprende o suficiente para começar a responder perguntas.

Gostaríamos de acreditar que nossos arquivos digitais oferecem provas de disputas legais e financeiras, mas isso só funciona se pudermos confiar nesses arquivos.Algo proporcionado pela “immutable blob storage API".

BlazingText

O algoritmo BlazingText do Amazon SageMaker ajuda a compreender as palavras, marcando o texto com dados que descrevem como a palavra foi usada anteriormente. Você insere texto bruto e recupera um vetor de significado semântico e classificação de entidade que pode ser usado para análise de sentimento. Se você quiser mais poder computacional, os algoritmos podem rodar mais rápido em máquinas com múltiplos núcleos e GPUs.

Webdam

Oferecendo armazenamento em nuvem seguro com um fluxo de trabalho moderado e organizado. As fotos e os trabalhos de arte chegam dos criadores como arquivos, mas eles se tornam "ativos" dentro do sistema à medida que passam pela cadeia de aprovação dos anúncios, sites e folhetos que definem uma marca. A API Webdam permite que você aproveite seu próprio código interno e confie no Webdam como o sistema de arquivos que armazena e organiza suas imagens.

Google Drive

Você pode, por exemplo, criar e compartilhar documentos do Google Docs com a extensa API REST do Google Drive. O alvo principal são empresas que desejam uma maneira fácil de gerenciar relatórios automatizados regulares (e outros documentos) em um local centralizado. A API do Google Drive permite criar aplicativos que aproveitam o armazenamento em nuvem do Google Drive. Você pode desenvolver aplicativos que se integram ao Google Drive e criar funcionalidades robustas em seu aplicativo usando a API do Google Drive.

Google AR

Se uma imagem vale mais que mil palavras, quanto valeria se você adicionasse a elas pequenas bolhas de texto e detalhes que expliquem exatamente o que estamos vendo? A Realidade Aumentada (AR) oferece a oportunidade de adicionar muitos desses detalhes animados em tempo real ao que quer que nossa câmera veja - e compartilhar esses objetos virtuais e diagramas com outras pessoas. O Google quer construir o backbone que suporte essas interações de RA.
